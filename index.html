How this works

Tap the launch button, then pick a game's .html file from Files.

If the game uses separate .js, .css, or image files, multi-select those along with the .html file — tap several items before confirming.

iOS shows Photo Library/Camera options first since images are allowed — tap Choose Files (or Browse) to get to your files.

Nothing is ever saved here. Each launch starts fresh, so you'll choose the file(s) again next time.
